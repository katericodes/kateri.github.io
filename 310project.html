NYC Airbnbs

Context

Founded in 2008, Airbnb allows hosts to share their spaces with guests traveling from all over the world. The dataset describes the details of Airbnb listings based in NYC, NY in 2019.

Objective

The goal of this project is to predict the prices of Airbnb listings using the given datasets.

Content

The dataset contains 39118 Airbnb listings (rows) described by the following attributes:

  • id: id of listing
  • name: name of listing
  • host_id: id of host that posted the listing
  • host_name: name of host
  • neighbourhood_group: NYC borough where listing is located
  • neighborhood: NYC neighborhood where listing is located
  • latitude: latitude coordinate where listing is located
  • longitude: longitude coordinate where listing is located
  • room_type: type of room
  • minimum_nights: the minimum of nights that a guest must stay at the listing
  • number_of_reviews: the total number of reviews that the listing has
  • last_review: date of the most recent review of the listing
  • reviews_per_month: average number of reviews received by the listing in a month
  • floor: the floor of the listing
  • noise (dB): the average sound level (in decibel)
  • price: the price of the listing (in US dollar)

Approach

  1. Data Cleaning / Wrangling: Column names, data types, and data formats were cleaned and standardized in Python using tools such as RegEx to prepare the data for analysis.
  2. Exploratory Data Analysis: Patterns in the data were visualized using data plots in R (ggplot) to observe any trends. Z-test was used to conduct hypothesis testing to determine if any observations were statistically significant based on p-value.
  3. ML Prediction: Linear regression and r-part models were used to make the prediction models for the dataset. Through an internal Kaggle competition, linear regression was determined to be the better model for my prediction.
